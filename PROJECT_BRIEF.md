# PROJECT BRIEF: BeautiPhi Medispa

## 1. BUSINESS BRIEF & CONTEXT
**Brand:** BeautiPhi Medispa
**Positioning:** Advanced skin/laser clinic with a luxury spa experience (NOT a basic beauty salon).
**Vibe:** Clinical authority + Luxury Experience + Seamless Booking. Comparable to high-end Harley Street clinics.
**Audience:** Women 30-60, results-driven, willing to invest in premium treatments.
**Key Services:** Advanced Skin (Hydrafacial, Peels), Laser Hair Removal, Body Contouring, Medical Skincare.
**Tone:** Refined, Intelligent, Warm, Confident.

## 2. TECHNICAL STACK REQUIREMENTS
- **HTML5:** Semantic structure for SEO (header, nav, main, section, footer).
- **CSS3:** Modern Flexbox/Grid. Use CSS Variables for all colors/fonts to ensure consistency.
- **JavaScript:** Vanilla JS (no heavy frameworks) for speed and simplicity.
- **Responsiveness:** Mobile-first approach.
- **Performance:** Optimized for fast loading (lazy loading placeholders, minified structure).
- **Hosting:** Static site ready for GitHub Pages.

## 3. DESIGN SYSTEM GUIDELINES
- **Layout:** Editorial, magazine-style. Ample whitespace.
- **Typography:** 
  - Headings: Elegant Serif (e.g., Playfair Display or similar Google Font).
  - Body: Clean Sans-Serif (e.g., Lato or Montserrat).
- **Color Palette:** Define CSS variables for a "Luxury Medispa" palette (e.g., Soft Gold, Charcoal, Clean White, Muted Sage).
- **UI Elements:** Sharp edges or subtle radius (no overly bubbly shapes). Minimalist buttons with hover effects.
